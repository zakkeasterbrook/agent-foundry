# Foundry SDK (alpha)

Toolkit for building Agent Foundry-compliant agents in TypeScript/Node.

## Planned modules
- **Manifest loader** – parse YAML definitions and inject runtime constraints.
- **Proof helper** – emit standard proof JSON with minimal boilerplate.
- **Budget guardrail** – track cumulative spend per mission and stop overages.
